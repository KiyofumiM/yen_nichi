# 縁日(Yen Nichi)：日本語解説
縁日(Yen Nichi)は外国の方と日本の方が、伝えたい言葉で繋がるためのSNSです。

主な機能は、下記のとおりです

## １．言葉を広げる
伝えたい言葉とその言葉のイメージで、伝えたいメッセージを伝えます
言葉と画像を一緒に伝えることで、より相手に伝わりやすくなります。
例えば、戦争中の画像と、「戦争の悲惨さ」。言葉だけでなく、画像があることで、より戦争の悲惨さが伝わりますね。
以上が言葉を広げるの説明です。

## ２．イベントの共有
互いに大切にしている行事や、行われるイベントを紹介しあいましょう。
互いの文化の理解が進むことで、より相手の考えやその国のことがより深く知れますね。
そして、理解が進むことで、世界がより近くなるといいですね。
以上がイベントの共有の説明です。

## ３．掲示板
掲示板を利用して、お互いが持っている情報を積極的に共有していきましょう。
互いの知識が交わることで、より良い情報が生まれます。
以上が掲示文の説明です。

以上、上記の３点が、縁日の主な機能です。

それに加え、下記の機能もあります。
### ・表示言語の切り替え
ブラウザの言語設定を切り替えることで、表示する言語を切り替えることができます。
切り替えられる言語は、現在、日本語、英語、ベトナム語、ポルトガル語、フランス語、台湾語、韓国語、中国語、の８言語です。
要望があれば、その他言語も追加していく予定です。

### ・お問い合わせ機能
お問い合わせフォームもあり、改善要望などを、製作者にメールで送信することも出来ます。
送られてきた内容は、内容を検討の上、可能な限り縁日に反映させる予定です。

### ・使い方説明機能
使い方がわからなければ、ナビバーにある、使い方ボタンから使い方ページを見てください。
ログイン時、ログアウト時の画面、各動画にて、縁日の使い方を説明しています。
使い方がわからなくなった際は、ぜひ使い方説明画面にて、動画を見てください。
また、搭載している動画について、
・この部分がわかりにくい
・もっとこうした方がわかりやすい
などといった意見があれば、お問い合わせ画面にて、要望を送ってください。
内容を検討の上、可能な限り動画や画像に反映させる予定です。

以上が、縁日の説明となります。
縁日のURLは、 https://m3cc3ict.mydns.jp/portfolio/ となっています。
よろしければ、一度上記のリンクから縁日を使ってみてはいかがでしょうか。
